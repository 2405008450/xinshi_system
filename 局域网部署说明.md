# 局域网部署说明

## 网络配置信息
- **服务器IP地址**: 192.168.31.125
- **子网掩码**: 255.255.255.0
- **默认网关**: 192.168.31.1

## 访问地址
- **前端访问地址**: http://192.168.31.125:3000
- **后端API地址**: http://192.168.31.125:8000

## 启动步骤

### 1. 启动后端服务
在项目根目录下，双击运行 `start_backend.bat`

或者在命令行中执行：
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. 启动前端服务
在 `frontend` 目录下，双击运行 `start_frontend.bat`

或者在命令行中执行：
```bash
cd frontend
npm run dev
```

## 局域网内其他设备访问

局域网内的其他设备可以通过以下地址访问系统：
- 访问地址: http://192.168.31.125:3000

## 防火墙设置

如果无法访问，请确保 Windows 防火墙允许以下端口：
- 端口 8000 (后端API)
- 端口 3000 (前端服务)

可以在 Windows Defender 防火墙中添加入站规则：
1. 打开"Windows Defender 防火墙"
2. 点击"高级设置"
3. 点击"入站规则" -> "新建规则"
4. 选择"端口" -> 下一步
5. 选择"TCP"，输入特定本地端口：3000,8000
6. 选择"允许连接"
7. 全部勾选（域、专用、公用）
8. 输入名称："新实系统局域网访问"

## 注意事项

1. 确保电脑的IP地址保持 192.168.31.125 不变（可以在路由器中设置IP地址保留）
2. 如果IP地址变化，需要修改以下文件：
   - `frontend/vite.config.js` 中的 `target` 地址
3. 启动服务后，确保两个服务都正常运行再访问前端页面
